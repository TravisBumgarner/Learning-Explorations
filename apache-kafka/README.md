# Setup

### Frontend

1. `cd frontend && npm i`
2. `npm run sd:fe`

### Backend

1. `cd backend && npm i`
2. `npm run sd:be`

### Kafka Server

1. [Follow these instructions](https://kafka.apache.org/quickstart), specifically Steps 1 and 2 to start Zookeeper and Kafka.
### Kafka Consumer

1. `cd backend` (`npm i` if you didn't already)
2. `npm run sd:kafka-consumer`
